# Crowdsale

## Overview

This is a crowdsale contract for Snarfcoin on the [Ethereum](https://www.ethereum.org/) blockchain.

## Requirements

* [Node.js](https://nodejs.org/en/) >= 8.9.4
* [Hardhat](https://hardhat.org/) >= 2.0.0

## Installation

```bash
npm install
```
